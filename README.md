# RentinBO

## 1. Introduction

In case .ipynb files are not rendered correctly just copy-paste their  raw link to:
https://nbviewer.jupyter.org/.

The RentinBo project I here present is intended to be a demostration of the workflow behind the application of a Machine Learnign algorithm.
The scope of this project is to train a classification algorithm, with real data from current housing situation in Bologna, to predict the possible range of euros that can be asked to rent a single or double room.
